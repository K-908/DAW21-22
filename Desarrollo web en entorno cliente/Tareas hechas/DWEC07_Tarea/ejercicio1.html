Ejercicio 1

Buscar APIs libres que te ofrezcan unos datos tanto en formato XML como en formato JSON. Debes detallar qué tipos de datos te ofrecen páginas e indicar claramente su URL. Ten en cuenta que vas a necesitar una que ofrezca datos en formato XML y otra en formato JSON.

random-data-api.com

API con datos en JSON y XML

API que sirve para generar datos aleatorios. Puede generar datos de direcciones, bancos, nombres etc... En nuestro caso, usaremos la de coches

jokeapi.dev

API con datos en JSON y XML

Jokeapi.dev es una API pública que genera chistes en 6 idiomas distintos. Tiene varias opciones para elegir el tipo de chiste que queremos que muestre y podemos elegir que nos devuelva los datos en JSON, XML, YAML o texto plano.
